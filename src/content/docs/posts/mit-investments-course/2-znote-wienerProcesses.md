## 一句話總結

維納過程（Wiener process）是「隨機漫步」的連續時間版本，用來描述股價無法預測的隨機波動。它是 Black-Scholes 選擇權定價的數學基石。

---

## 1. 基本維納過程 dz

在一小段時間 ∆t 內，z 的變化量定義為：

$$\Delta z = \varepsilon \sqrt{\Delta t}, \qquad \varepsilon \sim N(0,1)$$

**兩個核心性質：**

1. ε 是從標準常態分布抽出的隨機值 → 每一步的方向、大小都是隨機的。
2. 不同時間區間的 ∆z 彼此**獨立** → z 服從**馬可夫過程**（只看現值，不看歷史路徑）。

**∆z 的分布**（因為 ε 平均 0、標準差 1）：

| 統計量 | 值 | 怎麼算來的 |
|---|---|---|
| 平均 | 0 | $E[\varepsilon]\sqrt{\Delta t}=0$ |
| 標準差 | $\sqrt{\Delta t}$ | $\text{sd}(\varepsilon)\sqrt{\Delta t}=\sqrt{\Delta t}$ |
| 變異數 | $\Delta t$ | 標準差平方 |


> 直覺：時間拉長，標準差以 **√ 時間**增加，不是線性增加。這就是「亂度隨時間擴散」的特徵。

---

## 2. 拉長到一段時間 T

把長度 T 切成 N 小段，每段 ∆t：

$$N = \frac{T}{\Delta t}, \qquad z(T)-z(0)=\sum_{i=1}^{N}\varepsilon_i\sqrt{\Delta t}$$

因為各 εᵢ 獨立，總和仍是常態分布。**變異數可以直接相加**：

$$\text{Var}[z(T)-z(0)] = N\cdot\Delta t = \frac{T}{\Delta t}\cdot\Delta t = T$$

所以對整段 T：

- 平均 = 0
- **變異數 = T**
- **標準差 = √T**

> ⚠️ 講義 (8)(9) 式印成 √∆t、∆t，應為 **√T、T**（見下方數值例驗證；講義也說「z 在區間 T 的變化量變異數等於 T」）。

**數值例：** 設 T = 1 年，切成 N = 252 個交易日，∆t = 1/252。
每天 $\Delta z=\varepsilon\sqrt{1/252}$，每天標準差 ≈ 0.063。
整年變異數 = $252 \times (1/252)$ = **1** → 標準差 = $\sqrt{1}$ = **1**，與 $\sqrt{T}=\sqrt{1}$ 一致 ✓。
（不論切多細，加總後的變異數恆為 T，與 ∆t 無關。)

---

## 3. 廣義維納過程（加上漂移）

基本 dz 只有隨機、沒有方向。實際變數常有趨勢，所以加上漂移項：

$$dx = a\,dt + b\,dz$$

- **$a\,dt$（漂移項）**：確定性趨勢，每單位時間平均增加 $a$。若沒有 $b\,dz$，則 $x = x_0 + at$，是一條斜率 $a$ 的直線。
- **$b\,dz$（雜訊項）**：隨機波動，大小是 $dz$ 的 $b$ 倍。

離散版：$\Delta x = a\,\Delta t + b\,\varepsilon\sqrt{\Delta t}$

**∆x 在區間 T 的分布：**

| 統計量 | 值 |
|---|---|
| 平均 | $aT$ |
| 標準差 | $b\sqrt{T}$ |
| 變異數 | $b^2 T$ |

→ 期望漂移率 = $a$，變異率 = $b^2$。

---

## 4. 為什麼股價不能直接用廣義維納過程？

關鍵：投資人要求的是**百分比報酬**，與股價水準無關。
股價 \$10 要求年化 14%，股價 \$50 也要求 14%。
若用固定 a（絕對漲跌金額固定），就違反這點。

**修正**：把漂移與波動都改成「與股價成比例」：

$$dS = \mu S\,dt + \sigma S\,dz \qquad\Longleftrightarrow\qquad \frac{dS}{S} = \mu\,dt + \sigma\,dz$$

- $\mu$ = 期望報酬率（如 12%/年）
- $\sigma$ = 波動率（如 15%/年）

這就是**幾何布朗運動（Geometric Brownian Motion）**，最廣泛使用的股價模型。

**離散版（報酬率形式）：**

$$\frac{\Delta S}{S} = \mu\,\Delta t + \sigma\varepsilon\sqrt{\Delta t} \;\sim\; N\!\left(\mu\Delta t,\ \sigma\sqrt{\Delta t}\right)$$

- 期望報酬 = $\mu\Delta t$
- 隨機成分 = $\sigma\varepsilon\sqrt{\Delta t}$，其變異數 = $\sigma^2\Delta t$
- 即 $\sigma\sqrt{\Delta t}$ 是短期報酬的標準差（這就是波動率 $\sigma$ 的定義）

**數值例（換算日報酬）：** µ = 12%/年、σ = 15%/年，∆t = 1/252：
- 日期望報酬 = $0.12 / 252 \approx$ **0.00048**
- 日標準差 = $0.15 / \sqrt{252} \approx$ **0.0094**（≈ 0.94%）

> 注意：波動率隨時間是 **√時間** 縮放（$\sigma\sqrt{\Delta t}$），期望報酬是**線性**縮放（$\mu\Delta t$）。時間越短，隨機波動相對於趨勢越占主導 → 短線幾乎是純隨機。

---

## 重點回顧

- 維納過程 = 連續時間的隨機漫步，標準差 $\propto \sqrt{\text{時間}}$。
- 馬可夫性質 ⟺ 弱式市場效率（現價已含過去所有價格資訊，技術分析難有超額報酬）。
- 廣義維納過程 = 趨勢 $a$ + 雜訊 $b$。
- 股價模型把 $a$、$b$ 換成 $\mu S$、$\sigma S$，得到幾何布朗運動 $\dfrac{dS}{S}=\mu\,dt+\sigma\,dz$。
