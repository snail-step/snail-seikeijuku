import { Aside, Badge, Card, CardGrid, Icon, Tabs, TabItem, LinkCard, LinkButton, Steps, FileTree } from '@astrojs/starlight/components';

這是 `Starlight` (UI 套件工具) 內建元件的使用範例，複製貼上後依需求修改。

**注意：這些元件需要寫在 `.mdx` 檔案裡（不能用在 `.md`），且要在檔案最上方 import，如本檔案第 9 行：**

`import { Aside, Badge, Card, CardGrid, Icon, Tabs, TabItem, LinkCard, LinkButton, Steps, FileTree } from '@astrojs/starlight/components';`

## Aside（提示框）

```mdx
<Aside type="note">這是一般提示。</Aside>
<Aside type="tip" title="自訂標題">這是 tip 類型，可自訂標題。</Aside>
<Aside type="caution">這是注意事項。</Aside>
<Aside type="danger">這是危險警告。</Aside>
```

<Aside type="note">這是一般提示。</Aside>
<Aside type="tip" title="自訂標題">這是 tip 類型，可自訂標題。</Aside>
<Aside type="caution">這是注意事項。</Aside>
<Aside type="danger">這是危險警告。</Aside>

也可以不用 import，直接用 Markdown 指令語法（在 .md / .mdx 都能用）：

```md
:::note
這是一般提示。
:::

:::tip[自訂標題]
這是 tip 類型。
:::
```

:::note
這是一般提示。
:::

:::tip[自訂標題]
這是 tip 類型。
:::
